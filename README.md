# Keeper

Keeper is a client-server system that allows the user to securely and securely store logins, passwords, binary data and other private information.

[![Go Report Card](https://goreportcard.com/badge/github.com/srg-bnd/keeper?style=flat-square)](https://goreportcard.com/report/github.com/srg-bnd/keeper)

## Features

### Server

- User registration, authentication, and authorization
- Storing private data
  - login/password pairs
  - arbitrary text data
  - arbitrary binary data
  - bank card details
- Data synchronization between multiple authorized clients of the same owner
- Transfer of private data to the owner upon request.

### Client (CLI)

- Authentication and authorization of users on a remote server
- Access to private data on request

Platforms: Windows, Linux, and Mac OS
